# PhoneBook

PhoneBook is a .NET interview assignment for building a DDD-oriented REST API, beginning with a clean solution skeleton and layered architecture.

## Architecture

- `PhoneBook.Domain`
- `PhoneBook.Application`
- `PhoneBook.Infrastructure`
- `PhoneBook.Api`
- `PhoneBook.Domain.UnitTests`
- `PhoneBook.Application.UnitTests`
- `PhoneBook.Infrastructure.UnitTests`
- `PhoneBook.Api.IntegrationTests`

## Build

```shell
dotnet build
```

## Test

```shell
dotnet test
```

Implementation will be added incrementally.
